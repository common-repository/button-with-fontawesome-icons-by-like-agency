== Description ==

This plugin allows you to add a widget with custom text, link and a FontAwesome icon. In order for this plugin to work, **please make sure you already use FontAwesome toolkit** in your theme.

List of features:

*   Choose FontAwesome icon as an image for the button.
*   Add your own text and title for the widget.
*   Select header tag for the title.
*   Customize your FontAwesome image (position, color and background color).
*   Add custom URL to use the widget as a link.
*   Add extra button displayed below, with your own text.
*   The widget has its own stylesheet included, however with just necessary CSS to be displayed properly, so it doesn't override the style for any theme you currently use. 
*   You can also add your own CSS for this plugin via its Settings page.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Appearance->Widgets and select 'Button with FontAwesome icons by LIKE.agency'
